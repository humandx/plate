Use this plugin to render a `<hr>` HTML tag.

Type three dashes (`---`) or three underscores and a space (`___ `) at the beginning of a new line and it will magically transform to a horizontal rule.

### Installation

```bash npm2yarn
npm install @humandx/plate
# or
npm install @humandx/plate-horizontal-rule
npm install @humandx/plate-ui-horizontal-rule
npm install @humandx/plate-autoformat
```

### Usage

import { HorizontalRuleSandpack } from './HorizontalRuleSandpack';

<HorizontalRuleSandpack />
