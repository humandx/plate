The emoji plugin is using the [combobox plugin](docs/plugins/combobox).

### Installation

```bash npm2yarn
npm install @humandx/plate
# or
npm install @humandx/plate-combobox
npm install @humandx/plate-emoji
npm install @humandx/plate-ui-combobox
npm install @humandx/plate-ui-emoji
```

### Usage

import { EmojiSandpack } from './EmojiSandpack';

<EmojiSandpack />

### Options

```ts
interface EmojiPlugin<TData extends EmojiItemData = EmojiItemData> {
  trigger?: string;
  createEmoji?: CreateEmoji<TData>;
  emojiTriggeringController?: IEmojiTriggeringController;
  id?: string;
}
```
