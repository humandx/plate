The mention plugin is using the [combobox plugin](docs/plugins/combobox).

### Installation

```bash npm2yarn
npm install @humandx/plate
# or
npm install @humandx/plate-ui-combobox
npm install @humandx/plate-mention
npm install @humandx/plate-ui-mention
```

### Usage

import { MentionSandpack } from './MentionSandpack';

<MentionSandpack />

### Options

```ts
interface MentionPlugin<TData extends Data = NoData> {
  createMentionNode?: CreateMentionNode<TData>;
  id?: string;
  insertSpaceAfterMention?: boolean;
  trigger?: string;
}
```
