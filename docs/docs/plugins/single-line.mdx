Allows you to force the editor to only have one line.

### Installation

```bash npm2yarn
npm install @humandx/plate
# or
npm install @humandx/plate-break
```

### Usage

import { SingleLineSandpack } from './SingleLineSandpack';

<SingleLineSandpack />
