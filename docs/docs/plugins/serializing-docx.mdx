### Installation

```bash npm2yarn
npm install @humandx/plate @humandx/plate-juice
# or
npm install @humandx/plate-serializer-docx @humandx/plate-juice
```

Given pasted HTML, `@humandx/plate-juice` is used to inline CSS properties into the `style` attribute.
